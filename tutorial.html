Functions as shortcuts

						/* helper function for getting elements by Id */
						var $ = function(Id) {
								return document.getElementById(Id);
							}

I don't know exactly why I think this is so cool, especially since it's about the most simple thing one can do with Javascript. But I do. It might be that I always thought that "$" was some kind of magic symbol JQuery provided-- I didn't realize that Javascript also let me define "$" in a way that is useful to me.

So when I call something like $('foo') that acts as a shortcut for the long-winded document.getElementById() method, it just gives me code monkey warm fuzzies all over the place. I know, I need to get out more.

But I digress. The way it works is you define a function that wraps some functionality, using "$" as the function name, instead of something more verbose:

								var $ = function() {
										// do something useful...
										return "baz!";
									}
									
									/*  call our useful function... */
									var foo = $();
									alert(foo); // displays "baz!"

And that's it. to use this shortcut, just call the $() function and soak up the concise goodness.
